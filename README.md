–Ü–Ω–æ–¥—ñ –ø–æ—Ç—Ä—ñ–±–Ω–æ –ø–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏, —á–∏ –≤–∏–∫–æ–Ω—É—î—Ç—å—Å—è —è–∫–∞—Å—å —É–º–æ–≤–∞. –£ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ñ —Ç–∏ –æ—Ç—Ä–∏–º–∞—î—à –æ–¥–Ω—É –∑ –¥–≤–æ—Ö –≤—ñ–¥–ø–æ–≤—ñ–¥–µ–π: `—Ç–∞–∫, –≤—ñ—Ä–Ω–æ` –∞–±–æ `–Ω—ñ, –Ω–µ –≤—ñ—Ä–Ω–æ`.

–î–ª—è —Ç–∞–∫–∏—Ö –ø–µ—Ä–µ–≤—ñ—Ä–æ–∫ –≤ Javascript –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è —Ç–∏–ø `boolean`. –í—ñ–Ω –º—ñ—Å—Ç–∏—Ç—å —É—Å—å–æ–≥–æ –¥–≤–∞ –∑–Ω–∞—á–µ–Ω–Ω—è: `true` –∞–±–æ `false` (—ñ—Å—Ç–∏–Ω–∞ —á–∏ –±—Ä–µ—Ö–Ω—è).

–ù–∞–ø—Ä–∏–∫–ª–∞–¥:


### –ü–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è —á–∏—Å–µ–ª

–î–ª—è –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è —á–∏—Å–µ–ª —É Javascript –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è —Ç–∞–∫—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏: ``===`, `!==`, `>`, `<`, `>=` —Ç–∞ `<=`.

–†–æ–∑–≥–ª—è–Ω–µ–º–æ –ø—Ä–∏–∫–ª–∞–¥–∏:

```css

2 < 3, // true
2 < 1, // false
2 < 2, // false
);

// `>=` –ø–µ—Ä–µ–≤—ñ—Ä—è—î, —á–∏ –∑–Ω–∞—á–µ–Ω–Ω—è –ª—ñ–≤–æ—Ä—É—á –±—ñ–ª—å—à–µ –∞–±–æ –¥–æ—Ä—ñ–≤–Ω—é—î –∑–Ω–∞—á–µ–Ω–Ω—é –ø—Ä–∞–≤–æ—Ä—É—á

1 >= 2, // false

// `<=` –ø–µ—Ä–µ–≤—ñ—Ä—è—î, —á–∏ –∑–Ω–∞—á–µ–Ω–Ω—è –ª—ñ–≤–æ—Ä—É—á –º–µ–Ω—à–µ –∞–±–æ –¥–æ—Ä—ñ–≤–Ω—é—î –∑–Ω–∞—á–µ–Ω–Ω—é –ø—Ä–∞–≤–æ—Ä—É—á
```

### –ü–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è —Ä—è–¥–∫—ñ–≤

–†—è–¥–∫–∏ –≤ Javascript –ø–æ—Ä—ñ–≤–Ω—é—é—Ç—å—Å—è –ª—ñ—Ç–µ—Ä–∞ –∑–∞ –ª—ñ—Ç–µ—Ä–æ—é –≤ –∞–ª—Ñ–∞–≤—ñ—Ç–Ω–æ–º—É –ø–æ—Ä—è–¥–∫—É. –ü–æ—è—Å–Ω–∏–º–æ:

2. –Ø–∫—â–æ –≤–æ–Ω–∏ —Ä—ñ–∑–Ω—ñ, —Ç–æ –±—ñ–ª—å—à–∏–π —Ç–æ–π —Ä—è–¥–æ–∫, —É —è–∫–æ–º—É –ø–µ—Ä—à–∞ –ª—ñ—Ç–µ—Ä–∞ —Å—Ç–æ—ó—Ç—å –ø—ñ–∑–Ω—ñ—à–µ –≤ –∞–ª—Ñ–∞–≤—ñ—Ç—ñ.
3. –Ø–∫—â–æ –ø–µ—Ä—à—ñ –ª—ñ—Ç–µ—Ä–∏ –æ–¥–Ω–∞–∫–æ–≤—ñ, —Ç–æ –ø–æ—Ä—ñ–≤–Ω—é—é—Ç—å—Å—è –¥—Ä—É–≥—ñ –ª—ñ—Ç–µ—Ä–∏ –∑–∞ —Ç–∏–º —Å–∞–º–∏–º –ø—Ä–∏–Ω—Ü–∏–ø–æ–º —Ç–æ—â–æ.
4. –ü–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è –ø—Ä–æ–¥–æ–≤–∂—É—î—Ç—å—Å—è, –ø–æ–∫–∏ —á–µ—Ä–≥–æ–≤–∞ –ª—ñ—Ç–µ—Ä–∞ –æ–¥–Ω–æ–≥–æ –∑ —Ä—è–¥–∫—ñ–≤ –Ω–µ –≤–∏—è–≤–∏—Ç—å—Å—è –±—ñ–ª—å—à–æ—é –∑–∞ –≤—ñ–¥–ø–æ–≤—ñ–¥–Ω—É –ª—ñ—Ç–µ—Ä—É –≤ —ñ–Ω—à–æ–º—É —Ä—è–¥–∫—É.
5. –Ø–∫—â–æ –≤ –æ–¥–Ω–æ–º—É –∑ —Ä—è–¥–∫—ñ–≤ –∑–∞–∫—ñ–Ω—á–∏–ª–∏—Å—è –ª—ñ—Ç–µ—Ä–∏, –∞ –≤ –¥—Ä—É–≥–æ–º—É –Ω—ñ, —Ç–æ –¥—Ä—É–≥–∏–π –±—ñ–ª—å—à–µ.
6. –Ø–∫—â–æ –≤—Å—ñ –ª—ñ—Ç–µ—Ä–∏ –æ–¥–Ω–∞–∫–æ–≤—ñ, —Ç–æ —Ä—è–¥–∫–∏ —Ä—ñ–≤–Ω—ñ.

–ù–∞–ø—Ä–∏–∫–ª–∞–¥:


**–ó–≤–µ—Ä–Ω–∏ —É–≤–∞–≥—É:** –≤ Javascript –º–∞–ª—ñ –ª—ñ—Ç–µ—Ä–∏ –±—ñ–ª—å—à–µ –≤–µ–ª–∏–∫–∏—Ö, —Ç–æ–º—É –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è `'a' > 'Y'` –ø–æ–≤–µ—Ä—Ç–∞—î `true`:

### –û–ø–µ—Ä–∞—Ç–æ—Ä –∑–∞–ø–µ—Ä–µ—á–µ–Ω–Ω—è (!)

–©–æ–± –æ—Ç—Ä–∏–º–∞—Ç–∏ –ø—Ä–æ—Ç–∏–ª–µ–∂–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è –∑–º—ñ–Ω–Ω–æ—ó –∞–±–æ –≤–∏—Ä–∞–∑—É, –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞—Ç–∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä –ª–æ–≥—ñ—á–Ω–æ–≥–æ –∑–∞–ø–µ—Ä–µ—á–µ–Ω–Ω—è `!`

### –õ–æ–≥—ñ—á–Ω–µ AND (&&)

–Ø–∫—â–æ –ø–æ—Ç—Ä—ñ–±–Ω–æ –ø–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏, —â–æ –≤–∏–∫–æ–Ω—É—é—Ç—å—Å—è –≤—Å—ñ —É–º–æ–≤–∏, –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞—Ç–∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä `&&` ‚Äî –ª–æ–≥—ñ—á–Ω–µ AND.
